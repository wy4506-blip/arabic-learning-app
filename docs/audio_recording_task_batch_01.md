# 音频补齐任务清单 Batch 01

本批先处理第一优先级内容，原因很明确：这批内容最直接影响初学者的主学习路径，出现频率高、复用率高、点击播放的概率也最高。包含 1 条字母标准发音、9 条课程例句，以及 20 条最核心的语法基础项，适合优先安排真人配音或 AI 批量生成。

统一录音标准：

- 发音自然
- 节奏平缓
- 不要过快
- 适合阿拉伯语初学者
- 句子边界清楚
- 不要明显机器腔

建议执行方式：

- 本批统一先出 slow 版本
- 单条单文件，文件名尽量保持稳定，方便后续批量导入
- 语气统一为温和、平缓、鼓励式，不要夸张演绎
