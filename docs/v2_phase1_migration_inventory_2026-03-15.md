# V2 Phase 1 课程迁移盘点

日期：2026-03-15

## 1. 目的

本盘点只处理 Phase 1，也就是：

- 字母
- 音符
- 拼读入门

目标不是立刻定最终 UI，而是先把 5 个问题填清楚：

1. 当前 lesson 是什么
2. 它的真实目标是什么
3. 现有录音是否可复用
4. completion rule 应该怎么定义
5. 是保留、拆分还是补桥接课

## 2. 当前真实内容源

先明确：当前 Phase 1 不是一套完整的 V2 lesson，而是由 4 类内容拼出来的。

### 2.1 当前可视为主线的内容

- 7 个字母分组
- 单字母学习页
- 单字母听读页
- 单字母书写页

### 2.2 当前更像 drill 的内容

- Level 1：字母识别
- Level 2：字母辨析
- Level 3：基础发音
- Level 4：13 音位

### 2.3 当前更像参考资料的内容

- 字母总表
- 字母位置变化
- 连写规则
- 音符与发音规则

### 2.4 当前完成口径的真实状态

当前字母阶段完成逻辑不是 V2 lesson 完成逻辑，而是：

- 单字母 viewed + listen 即可计入 group 完成
- write 当前不进入 group 完成判定
- 4 个练习关卡没有稳定的课程完成状态，只是练习入口
- grammar 页面更像资料，不是 lesson

这意味着：

- 当前 Phase 1 已有内容骨架
- 但 lesson 边界、达标判定和阶段分流都还没收口

## 3. 录音可复用性结论

### 3.1 可明确复用

- 主 manifest 已有 alphabet scope 的字母本体音频
- 主 manifest 已有 alphabet scope 的例词音频
- 主 manifest 已有 alphabet scope 的 13 音位读音音频
- grammar / pilot batch 已有少量慢速示范音频，可作为 Phase 1 补桥接素材

### 3.2 可复用但覆盖不均

- 字母本体：现有覆盖较强，但以主 manifest 的 normal 为主
- 例词：已有 normal 和部分 slow
- 13 音位：已有大量 normal，但 slow 覆盖不足
- harakat 示例：已有少量明确落地的 slow 样本，例如 بَ、أَيْ

### 3.3 当前不应误判为“已完整可用”

- 音符和拼读入门还没有成体系 lesson 音频包
- 许多资源是“可播放素材”，还不是“已组织好的 lesson 音频链”

## 4. 迁移判断总则

### 4.1 字母分组主线大体保留

7 个分组本身已经接近 lesson 粒度，不建议先彻底打散成 28 节独立主课。

原因：

- 当前首页与字母阶段心智已围绕 group 建立
- 7 组更符合新手记忆负担
- 现有 review / progress / smoke 保护面也围绕 group

### 4.2 单字母页不应继续当“lesson”

单字母页更适合作为 group lesson 内的 item drill / detail node，而不是课程主单元。

### 4.3 音符与拼读必须补桥接课

当前“13 音位”和 grammar harakat 页面能提供素材，但还不是面向新手的 lesson。

因此：

- 核心短元音必须单独做桥接课
- 长音与软音必须单独做桥接课
- 连写与断连需要从规则页转成识别型桥接课
- 拼读入门目前基本缺正式 lesson，必须新增

## 6. 建议新增的 Phase 1 桥接课

这是当前最关键的缺口。

### 6.1 P1-B01 核心短元音入门

- 当前来源：音符与发音规则页 + 13 音位资源
- 真实目标：用户能把 fatha / kasra / damma 和 a / i / u 对起来
- 现有录音：可部分复用，至少已有 بَ 类示例；需补成系统化最小集合
- completion rule：核心路径完成 + 至少 3 个短元音例子全部配对正确 + 1 次听音选符成功
- 迁移方式：新增桥接课

### 6.2 P1-B02 静符与最短闭音节

- 当前来源：音符规则页 + 13 音位 sukun 资源
- 真实目标：用户知道“有元音”和“无元音”差别，并能读最短闭音节
- 现有录音：可复用部分；慢速和最短对比读法需补
- completion rule：核心路径完成 + 2 组开闭对比判断正确 + 1 个最短拼读成功
- 迁移方式：新增桥接课

### 6.3 P1-B03 长音与半元音桥接

- 当前来源：字母组 7 + 13 音位中的 long_a / long_i / long_u / soft_ay / soft_aw
- 真实目标：用户知道 ا / و / ي 不只是在字母表里出现，也会参与长音和软音
- 现有录音：可复用较多，但 slow 应补齐
- completion rule：核心路径完成 + 至少 3 个长音 / 软音例子识别正确 + 能区分一组短音 / 长音
- 迁移方式：新增桥接课

### 6.4 P1-B04 连写与断连桥接

- 当前来源：字母组 3、字母位置变化、连写规则页
- 真实目标：用户在读最短词时知道什么时候会断开，什么时候会连写
- 现有录音：不强依赖录音，重点是视觉识别；可补少量提示音
- completion rule：核心路径完成 + 至少 3 个断连判断正确 + 1 个最短词切分正确
- 迁移方式：新增桥接课

### 6.5 P1-B05 最短拼读入门

- 当前来源：字母组 + 音符桥接素材 + 例词资源
- 真实目标：用户第一次完成“字母 + 音符 -> 最短可读组合”的真正拼读
- 现有录音：有素材，但还没组织成课；需要最少量补录来形成节奏
- completion rule：核心路径完成 + 至少 3 个 CV / CVC 组合拼读成功 + 1 次听音选拼写通过
- 迁移方式：新增桥接课

## 7. 第一轮推荐收口方案

### 7.1 课程主线

建议第一轮先把 Phase 1 收成 12 节左右的 V2 lesson：

1. 7 个字母分组 lesson 保留
2. 4 个桥接课新增
3. 1 个 Phase 1 阶段回顾课新增

### 7.2 不建议直接当 lesson 的内容

- 单字母首页
- 单字母听读页
- 单字母书写页
- 4 个 drill level
- 3 个 grammar 速查页

这些内容应转为：

- lesson 内 block
- 巩固 / recap 题池
- 参考资源

### 7.3 第一轮最需要修正的 completion rule

当前最不合理的一点是：

- 字母 group 完成只看 viewed + listen
- write 完全不影响 group 完成
- 音符 / 拼读根本没有课程完成口径

建议第一轮改为：

- group lesson：看字识别 + 听音识别为主完成
- write：先不做硬门槛，但进入加练或桥接课
- 音符 / 拼读：必须补独立 completion rule
- 13 音位：不再作为首轮主线完成门槛

## 8. 给后续 lesson 卡的直接结论

后续继续填逐课迁移卡时，可直接沿用以下判断：

1. 7 个字母分组先按“保留补标准”处理。
2. 单字母页全部按“吸收入 lesson block”处理，不再单独算课。
3. 音符与拼读入门按“新增桥接课”处理。
4. 13 音位按“保留为 drill + 桥接课题池”处理。
5. 字母阶段结束条件不应再只靠 viewed + listen，需要由 group 主线 + 至少一节拼读桥接共同收口。
