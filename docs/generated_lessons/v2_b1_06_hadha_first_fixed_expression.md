# 1. Working title

This Is... Your First Fixed Expression

# 2. Core objective

Help the learner understand, build, and recall the fixed beginner expression pattern `هذا + noun` using already known real words.

# 3. Why this lesson exists after Stage A and Lesson 5

Stage A established that Arabic can be entered through real words, not abstract letter study alone. Lesson 5 then extended that trust by giving the learner a third owned word: `قلم`.

Lesson 6 exists to convert that growing word set into the learner's first true short expression success. The learner should feel that Arabic is moving forward from "I know some words" into "I can already read and say a tiny Arabic line."

This lesson should not feel like an early grammar lecture. Its job is to show that known words can be reused inside one stable beginner frame, so Arabic starts feeling connected and usable.

# 4. Learner-visible outcome

By the end of the lesson, the learner should feel:

- "I can already read a very short Arabic expression."
- "I can understand and build `هذا كتاب` and `هذا قلم`."
- "I can say one complete Arabic line, not just a single word."
- "Arabic is starting to feel usable, not just learnable."

# 5. Hidden self-learning outcome

The lesson quietly teaches that beginner Arabic phrases can often be learned as stable chunks built from:

- a fixed frame
- a known word inside the frame
- repeated recognition and retrieval

It also introduces an important learning habit: when the learner already owns real words, phrase learning can begin by reusing known content instead of adding a large amount of new vocabulary or analysis.

# 6. Target knowledge scope

Primary target frame:

- `هذا + noun`
- beginner meaning support: `this is [noun]`

Primary target expressions:

- `هذا كتاب`
- `هذا قلم`

Low-stakes transfer expression:

- `هذا باب`

Scope boundary:

- The core objective is the fixed expression frame, not three separate new phrases.
- `هذا` should be taught as a stable beginner chunk inside a useful line, not as a grammar topic.
- Do not explain full demonstrative grammar, case, sentence theory, or definiteness.
- `هذا باب` can appear as a transfer or confidence check, but it should not become a third equally weighted target.

# 7. Input design

Input should make the expression feel immediately meaningful and beginner-usable.

Recommended input flow:

1. Known object reactivation
   - Briefly re-show `كتاب` and `قلم`
   - Remind the learner: these are words you already know

2. First modeled expression
   - Show `هذا كتاب`
   - Give audio
   - Give beginner meaning support: `this is a book`
   - Use a pointing/object context so the learner sees the line as a whole meaningful chunk

3. Second modeled expression
   - Show `هذا قلم`
   - Give audio
   - Give beginner meaning support: `this is a pen`
   - Keep the learner focused on the stable opening plus the known noun

4. Frame awareness without grammar load
   - Learner-facing line:
     "You already know the object word. Now Arabic can wrap it in a tiny useful frame."
   - The point is reuse, not explanation-heavy analysis

Input should avoid:

- grammar terminology
- broad sentence teaching
- adding more than one new frame
- new unknown nouns beyond low-stakes transfer

# 8. Recognition design

Recognition should prove that the learner can map the new fixed frame to meaning, sound, and known noun substitution.

Recommended recognition sequence:

1. Expression -> meaning
   - Show `هذا كتاب`
   - Ask learner to choose `this is a book`

2. Meaning -> expression
   - Prompt with `this is a pen`
   - Ask learner to choose `هذا قلم` from a very small set

3. Audio -> expression
   - Play `هذا كتاب` or `هذا قلم`
   - Ask learner to choose the matching Arabic line

4. Fixed-frame recognition
   - Show two short lines built from known nouns
   - Ask which line says `this is a pen` or `this is a book`
   - This should confirm that the learner is recognizing the whole expression, not only one word inside it

Recognition items should:

- keep choices very small
- reuse known nouns only
- make the fixed opening feel stable and reusable

# 9. Recall design

Recall must produce real evidence that the learner can rebuild one full expression, not only identify it.

Required recall-bearing step:

1. Meaning/context -> arranged expression
   - Prompt with `this is a pen` or show a pen cue
   - Learner must arrange `هذا قلم`
   - This should use `arrangeResponse`, because the lesson teaches a short phrase structure

Required reduced-support retrieval:

2. Meaning -> full expression recall
   - Prompt with `this is a book`
   - Learner must retrieve `هذا كتاب` with lighter support than the model stage

Optional low-stakes transfer check:

3. Transfer with known noun
   - Prompt `this is a door`
   - Learner recognizes or arranges `هذا باب`
   - This is a transfer signal, not the main mastery requirement

Recall must not be satisfied by:

- only recognizing `هذا كتاب` from options
- only repeating after the full answer remains visible
- only selecting the noun while ignoring the frame

# 10. Output design

Output should give the learner a clear first-line success moment.

Recommended output:

- one guided `speakResponse` for either `هذا كتاب` or `هذا قلم`

Purpose of output:

- help the learner feel they can already say one complete Arabic line
- reinforce the phrase as a meaningful chunk, not as isolated parts
- create a visible step up from single-word output in Lesson 5

Output boundary:

- do not demand spontaneous conversation
- do not grade detailed pronunciation nuance
- keep output to one stable line after enough guided exposure

# 11. Completion design

Completion should feel like a real beginner milestone: the learner has moved from owned words to a first usable expression frame.

The learner should complete the lesson only after demonstrating:

- they can recognize one `هذا + noun` expression from text
- they can recognize one target expression from audio
- they can correctly arrange one full target line
- they can recall one full target line with reduced support
- they can say one complete target line once

Completion message should make the gain visible:

- the learner can now read and say a tiny Arabic expression
- known words are becoming reusable inside a real frame
- the learner is ready for audio-first recognition of known content next

Completion should not claim:

- broad sentence production ability
- understanding of Arabic grammar theory
- mastery of all `هذا + noun` combinations

# 12. Completion contract

## mastery_status

`mastered` only if the learner successfully completes at least one recall-bearing full-expression build or retrieval and also succeeds on basic text/audio recognition for the same fixed frame.

`needs_review` if the learner can follow modeled examples but cannot reliably rebuild or recall one full `هذا + noun` expression.

## learning_evidence

Minimum acceptable evidence:

- recognized `هذا كتاب` or `هذا قلم` from meaning or text
- identified one target expression from audio
- correctly arranged one full target expression with `arrangeResponse`
- recalled one full target expression from meaning or context with reduced support

Stronger evidence:

- succeeds on both `هذا كتاب` and `هذا قلم`
- handles a low-stakes transfer check such as `هذا باب`
- completes one spoken output of a full line

## review_seed_candidates

Priority candidates:

- `fixed_frame_hadha_plus_noun`
- `expression_recall_hadha_kitab`
- `expression_recall_hadha_qalam`
- `noun_substitution_inside_hadha`

Fallback candidate if expression assembly is weak:

- `arrange_support_hadha_plus_noun`

## next_home_action

If `mastered`:

- continue to Lesson 7, where known words and expressions begin moving into audio-first recognition

If `needs_review`:

- schedule a short phrase review on `هذا + noun` before Lesson 7

# 13. Review seed logic

Seed 1: `fixed_frame_hadha_plus_noun`

- Trigger when the learner drops, misorders, or misreads the fixed opening
- Purpose: stabilize the expression frame as one reusable beginner chunk

Seed 2: `expression_recall_hadha_kitab`

- Trigger when `this is a book` does not retrieve `هذا كتاب`
- Purpose: repair full-line recall using the most familiar noun anchor

Seed 3: `expression_recall_hadha_qalam`

- Trigger when the learner knows `قلم` alone but fails to retrieve it inside the frame
- Purpose: strengthen reuse of a newly learned word inside a phrase

Seed 4: `noun_substitution_inside_hadha`

- Trigger when the learner keeps the frame but swaps `كتاب`, `قلم`, and `باب`
- Purpose: reinforce that the learner is managing both the stable frame and the selected noun

Seed priority rule:

- prioritize full-expression recall weakness first
- then frame instability
- then noun substitution confusion

Review boundary:

- review should keep the lesson centered on usable short lines
- do not split the experience into abstract grammar remediation

# 14. Paid-boundary note

This free lesson should feel substantial because it gives the learner the first true "I can already read and say something in Arabic" moment.

Paid expansion value remains intact because later content can still deepen:

- more fixed expressions across more situations
- broader daily-use phrase coverage
- richer speaking and listening contexts
- more flexible substitution patterns
- gradual movement from fixed chunks into wider sentence handling

Free value here is: "I can use my first tiny Arabic expression."
Paid value later is: "I can grow that into broader usable communication."

# 15. Implementation notes

- Keep exactly one formal lesson objective: understand, build, and recall the fixed expression frame `هذا + noun`.
- Treat `هذا` as a stable beginner chunk, not as a grammar lecture topic.
- Use `كتاب` and `قلم` as the main target nouns because they are already known and create a visible win fast.
- Use `باب` only as low-stakes transfer or contrast.
- Preferred V2 flow:
  - Input
  - Recognition
  - Recognition
  - Recall via `arrangeResponse`
  - Recall with reduced support
  - Output
  - Completion
- Because a phrase structure is being taught, include at least one required `arrangeResponse` item.
- Ensure completion cannot be earned through page traversal or recognition-only taps.
- Keep the opening frame visually stable across items so the learner notices reuse.
- Keep choice sets at 2-3 options maximum.
- Beginner meaning support can use natural English like `this is a book` even though deeper sentence analysis is intentionally postponed.
- Completion UI should visibly mark the new capability: the learner can now use known Arabic words inside a short fixed expression.
