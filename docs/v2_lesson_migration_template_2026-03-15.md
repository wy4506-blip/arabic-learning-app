# V2 Lesson 迁移模板（可执行版）

日期：2026-03-15

## 1. 用途

本模板用于把现有旧 lesson 迁移为符合 V2 标准的新 lesson。

适用场景：

- 旧 lesson 保留但补标准
- 旧 lesson 拆分成多个 V2 lesson
- 多个旧内容重组为一个能力导向 lesson
- 在旧内容前后补桥接课或巩固课

本模板的目标不是记录更多说明，而是确保每个 V2 lesson 都能回答 5 个问题：

1. 这节课到底教什么能力
2. 为什么这一课值得单独存在
3. 用户完成后系统拿什么判断达标
4. 这节课结束后如何分流到下一课或复习
5. 旧内容和旧音频哪些能直接继承

## 2. 使用顺序

建议按以下顺序填写：

1. 先填来源与迁移方式
2. 再写 can-do statement 和 2 到 4 个关键目标
3. 再设计 block 与核心互动
4. 最后写达标规则、复习挂接与音频继承策略

## 7. 文案检查

- 是否有明确目标文案
- 是否避免大段理论说明
- 是否反馈“学会了什么”而不是空泛鼓励
- 是否没有超出本课负担的百科补充

## 8. 验收清单

以下问题至少满足 7 项中的大多数，才算该 lesson 迁移合格：

1. 是否有明确能力目标
2. 是否有 2 到 4 个可验证关键目标
3. 是否能归入清晰的 lesson 类型
4. 是否有完整 block 结构
5. 是否能区分完成与达标
6. 是否已接入复习系统
7. 是否尽量继承了现有正确内容与可用音频

## 9. 快速示例

示例：把 U1L1 问候与基础表达拆成两个 V2 lesson。

- V2-L01A：高频问候识别与跟读
- can-do：你能听懂并跟读 3 个最常见问候表达
- 核心练习：听后选义、跟读一次、场景配对
- 达标：核心路径完成 + 2 个表达识别正确率达到 80%

- V2-L01B：基础回应与最短对话
- can-do：你能回应“你好吗”，并完成 2 轮最短对话
- 核心练习：回应选择、对话排序、跟读模仿
- 达标：核心路径完成 + 回应题正确率达到 80% + 1 次关键纠错通过
