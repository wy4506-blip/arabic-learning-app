## 1. 目标

本文件把当前 U1 的 4 节旧 lesson 拆成首批可执行的 V2 试点 lesson，用于：

- 先验证 V2 小课结构是否成立
- 先验证 lesson 完成、达标、复习挂接是否顺畅
- 先最大化复用旧内容与旧音频，不做无意义返工

本批不解决 Phase 1 与 Phase 2 的缺失问题。

前提假设：

- 用户已完成 V2 的字母、短元音、基础拼读入门
- 当前试点只验证 Phase 3 和部分 Phase 4 的重构方式

## 3. 执行原则

### 3.1 先拆结构，不先重写内容

- 旧词条、旧 pattern、旧 dialogue 只要正确，优先继承
- 旧 exercises 不直接原样搬用，而是改造成更轻的证据题
- 新增内容优先是 intro、过渡解释、recap 与桥接题

### 3.2 每节课只负责一个最小能力结果

- 不再把“词汇 + 对话 + 语法 + 练习”整包塞进一课
- 每课只保留 2 到 4 个关键 item
- 每课只采集最低可用能力证据

### 3.3 先把 U1 做成连贯路径

推荐顺序：

1. V2-U1-01 问候识别
2. V2-U1-02 回应与结束对话

4. V2-U1-04 我来自哪里
5. V2-U1-05 我是学生

## 4. 逐课迁移卡

### 4.1 V2-U1-01 问候识别

- 来源旧课：U1L1 问候与基础表达
- can-do：你能听懂并认出 3 个最常见的问候表达
- 关键 item：مَرْحَبًا / صَبَاحُ الْخَيْرِ / مَعَ السَّلَامَةِ
- 必保留旧内容：U1L1 vocabulary、pattern “مَرْحَبًا، كَيْفَ حَالُكَ؟” 的前半段
- intro：告诉用户这是一节“先听懂，不要求立刻完整回答”的轻课
- input：先听 3 个表达，再看阿语与中文对应
- explain：只解释这是固定表达，暂不拆词
- practice：听音选义、看阿语选中文、场景配对
- recap：你已经能听懂开场和结束语
- 达标：核心路径完成；3 个表达中至少 2 个听辨正确；至少做对 1 道场景配对题
- 复习种子：greeting_opening、greeting_closing、audio_recognition
- 下一步：进入 V2-U1-02

### 4.2 V2-U1-02 回应与结束对话

- 来源旧课：U1L1 问候与基础表达
- can-do：别人问“你好吗”时，你能选出合适回应并完成最短对话
- 关键 item：كَيْفَ حَالُكَ؟ / أَنَا بِخَيْرٍ، شُكْرًا. / مَعَ السَّلَامَةِ
- 必保留旧内容：U1L1 patterns、U1L1 dialogues
- intro：说明本课只做“会回应”，不讲复杂语法
- input：展示完整最短对话 1 次
- explain：说明“整句先整块记”
- practice：对话排序、回应选择、结束语选择、跟读一次
- recap：你已经能完成第一次最短阿语寒暄
- 达标：核心路径完成；回应题正确率不低于 80%；对话排序必须正确
- 复习种子：dialogue_reply、dialogue_sequence、audio_repeat
- 下一步：进入 V2-U1-03

### 4.3 V2-U1-03 我叫什么名字

- 来源旧课：U1L2 自我介绍
- can-do：你能回答“你叫什么名字”，并说出“我的名字是……”
- 关键 item：أَنَا / اِسْمِي / مَا اسْمُكَ؟ / اِسْمِي لِي.
- 必保留旧内容：U1L2 dialogue 的前两句、pattern “اِسْمِي لِي.”
- intro：明确本课只解决“名字介绍”
- input：先听问句，再听回答句
- explain：只解释 اِسْمِي 是固定介绍入口，不展开所有格理论
- practice：问答匹配、听后选句、把自己的名字代入模板
- recap：你已经能做第一句自我介绍
- 达标：核心路径完成；问答配对必须正确；至少 1 次模板代入成功
- 复习种子：self_intro_name、question_recognition、template_use
- 下一步：进入 V2-U1-04

### 4.4 V2-U1-04 我来自哪里

- 来源旧课：U1L2 自我介绍
- can-do：你能听懂“你来自哪里”，并用“我来自……”回应
- 关键 item：مِنْ / مِنْ أَيْنَ أَنْتَ؟ / أَنَا ... مِنَ الصِّينِ
- 必保留旧内容：U1L2 dialogue 的后两句、pattern “مِنْ أَيْنَ أَنْتَ؟ ”
- intro：说明本课只引入一个来源表达
- input：先整句输入，再拆出 مِنْ 的作用
- explain：只解释 مِنْ 表示“来自”
- practice：听问句选义、选出正确回应、来源句补全
- recap：你已经能说明来自哪里
- 达标：核心路径完成；问句识别正确率不低于 80%；来源句补全正确
- 复习种子：origin_question、origin_answer、particle_min
- 下一步：进入 V2-U1-05

### 4.5 V2-U1-05 我是学生

- 来源旧课：U1L2 自我介绍
- can-do：你能说“我是学生”，并初步知道 男学生 / 女学生 的对应关系
- 关键 item：طَالِبٌ / طَالِبَةٌ / أَنَا طَالِبٌ
- 必保留旧内容：word_student_intro、其阴性与复数信息、例句 “أَنَا طَالِبٌ مِنَ الصِّينِ.”
- intro：说明这是“表达 + 语法感”混合小课
- input：先展示整句，再展示 طالب / طالبة 对比
- explain：只解释“人物名词常需要同时记住阴性形式”
- practice：句义选择、阴阳配对、看中文选整句
- recap：你已经会说“我是学生”，并能区分男学生 / 女学生
- 达标：核心路径完成；طالبٌ 与 طالبةٌ 的区分题必须正确；整句表达正确率不低于 75%
- 复习种子：identity_sentence、gender_pair、noun_form_intro
- 下一步：进入 V2-U1-06

### 4.6 V2-U1-06 课堂指令先听懂

- 来源旧课：U1L3 课堂与学习用语
- can-do：你能听懂老师最常说的 2 个课堂指令
- 关键 item：اِقْرَأْ / كَرِّرْ / اِقْرَأْ مِنْ فَضْلِكَ.
- 必保留旧内容：U1L3 vocabulary 与第一句 pattern
- intro：强调本课目标是“先听懂老师在叫你做什么”
- input：音频先行，再给简短中文
- explain：只说明这类词是课堂高频动作指令
- practice：听指令选动作、听句选中文、场景点击
- recap：你已经能听懂“请读”“请重复”
- 达标：核心路径完成；两类指令都至少识别成功 1 次；总正确率不低于 80%
- 复习种子：classroom_command、listen_then_act
- 下一步：进入 V2-U1-07

### 4.7 V2-U1-07 我不懂，请重复

- 来源旧课：U1L3 课堂与学习用语
- can-do：不懂时，你能说“我不明白”或“请重复”
- 关键 item：أَنَا لَا أَفْهَمُ. / كَرِّرْ مِنْ فَضْلِكَ. / هَلْ تَفْهَمُ؟
- 必保留旧内容：U1L3 patterns、U1L3 dialogue 的后两句
- intro：明确这是“课堂自救表达”
- input：先给一段老师提问、学生求助的小对话
- explain：不展开动词变化，只告诉用户先整句会用
- practice：情景选回应、对话补全、跟读模仿
- recap：遇到听不懂时你已经不会卡死
- 达标：核心路径完成；情景回应题正确率不低于 80%；至少一次跟读完成
- 复习种子：help_request、classroom_survival_phrase、dialogue_completion
- 下一步：进入 V2-U1-08

### 4.8 V2-U1-08 教室物品识别

- 来源旧课：U1L4 教室里的物品
- can-do：你能认出并说出 3 个高频教室物品词
- 关键 item：قَلَمٌ / دَفْتَرٌ / سَبُّورَةٌ / كُرْسِيٌّ
- 必保留旧内容：U1L4 vocabulary、pattern “أَيْنَ الدَّفْتَرُ؟”
- intro：说明本课只做物品识别，不先讲指示代词规则
- input：物品词卡 + 例句轻输入
- explain：只说明这是教室最常见名词
- practice：看词选义、听词选物、问“本子在哪里”选理解
- recap：你已经能在教室里认词
- 达标：核心路径完成；4 个物品词中至少认对 3 个；位置问句理解正确
- 复习种子：classroom_noun、item_recognition、location_question_intro
- 下一步：进入 V2-U1-09

### 4.9 V2-U1-09 هذا / هذه 初识

- 来源旧课：U1L4 教室里的物品
- can-do：你能在最基础层面区分 هذا 和 هذه
- 关键 item：هٰذَا قَلَمٌ. / هٰذِهِ سَبُّورَةٌ. / masculine vs feminine noun
- 必保留旧内容：U1L4 patterns、grammarTitle 与 grammarExplanation
- intro：说明本课不讲完整规则，只建立最小区分
- input：先给两组整句对比
- explain：阳性名词前常见 هذا；阴性名词前常见 هذه
- practice：句子配对、名词分类、选出正确指示词
- recap：你开始有最基础的性别搭配感觉
- 达标：核心路径完成；هذا / هذه 选择题正确率不低于 80%；“سَبُّورَةٌ 是阴性名词”必须判断正确
- 复习种子：demonstrative_pair、gender_match、sentence_pattern
- 下一步：进入 V2-U1-10

### 4.10 V2-U1-10 U1 桥接巩固

- 来源旧课：U1L1-U1L4
- can-do：你能在一个连续路径里完成问候、介绍、课堂求助、物品识别的最短组合
- 关键 item：问候句、自我介绍句、求助句、一个教室名词
- 结构：不引入新内容，只做桥接
- practice：四步小任务串联；按顺序完成寒暄、名字、求助、识物
- 达标：核心链路走通；四类任务至少各对 1 个
- 复习种子：cross_lesson_bridge、mini_sequence
- 下一步：进入 V2-U1-11

### 4.11 V2-U1-11 U1 阶段回顾

- 来源旧课：U1L1-U1L4
- can-do：你能完成 U1 的最小综合判断，并知道下一步是继续还是先复习
- 结构：正式回顾课
- 核心检测：听辨问候、完成介绍、课堂求助、这个/这块 的区分
- 达标：四个目标各至少达标 1 项；总体现不稳则标记为 due_for_review
- 复习种子：formal_u1_review、weak_objective_redirect
- 下一步：Phase 3 后续 lesson 或首页复习分流

## 5. 旧音频处理建议

### 5.1 可直接复用

- U1L1 问候词和最短对话音频
- U1L2 自我介绍句和例句音频
- U1L3 课堂表达音频
- U1L4 物品词与例句音频

### 5.2 需要局部补录

- 每个 V2 lesson 的 intro 过渡提示
- bridge / recap 的收口语
- 若原音频没有 slow 版本，则优先补 slow，不先重录全部 normal

### 5.3 暂不建议重录

- 词条本身正确且已经进入旧资源清单的内容
- 仅因 lesson 被拆分而位置变化的例句音频

## 6. 数据挂接最低要求

每个 V2 lesson 至少要能产出：

- lessonId
- objectiveResults
- lessonStatus
- reviewSeeds
- nextLessonId
- recommendReviewAfterMinutes 或 recommendReviewAfterDays

## 7. 实施顺序建议

### 第一步

- 先做 V2-U1-01 到 V2-U1-05
- 先验证寒暄、自我介绍、身份表达的拆分方式

### 第二步

- 再做 V2-U1-06 到 V2-U1-09
- 先验证课堂表达和最小语法感的 lesson 粒度

### 第三步

- 最后接 V2-U1-10 与 V2-U1-11
- 用它们验证桥接课与正式回顾课的价值

## 8. 试点验收问题

1. lesson 时长是否明显降到 6 到 8 分钟区间
2. 用户是否更容易知道“我这节课到底学会了什么”
3. 完成与达标是否已经区分开
4. 旧内容与旧音频是否基本被保住
5. 首页是否能根据桥接课和回顾课结果给出合理下一步
