# 1. Curriculum overview

This V2 foundation track is a 16-lesson free curriculum for absolute beginners, organized into 4 stages of 4 lessons each:

- Stage A, Lessons 1-4: phonetics and script entry
- Stage B, Lessons 5-8: first usable words and expressions
- Stage C, Lessons 9-12: pattern awareness and very short text handling
- Stage D, Lessons 13-16: self-learning transfer and graduation

The track is designed to create visible beginner wins without trying to deliver full beginner Arabic coverage. By the end of Lesson 16, the learner should be able to:

- approach Arabic script with less fear and basic right-to-left confidence
- use diacritized beginner forms as reading support
- recognize, recall, and say a small but meaningful starter set of words and expressions
- notice that Arabic learning involves form, sound, meaning, and pattern, not only translation
- handle tiny guided beginner materials
- study one new word and one new phrase with a repeatable process
- understand why review matters and feel ready for paid continuation

Planned free-track value boundary:

- Free track gives: script entry, starter decoding, first usable language, first pattern awareness, first short-text handling, first self-learning method
- Paid track keeps: broader vocabulary, wider scene coverage, deeper listening, broader grammar, larger reading corpus, richer review personalization, and continuity over time

Stage milestones:

- End of Stage A: "Arabic is not as scary as I thought."
- End of Stage B: "I can already read and say some Arabic."
- End of Stage C: "I can see Arabic patterns and start handling them."
- End of Stage D: "I now know how to keep learning."

# 2. Why this structure aligns with authoritative beginner Arabic pedagogy at a high level

This plan follows the broad progression commonly used by respected beginner Arabic programs without copying any protected lesson wording or textbook sequencing.

- It starts with orthographic shock reduction. Strong beginner Arabic teaching usually addresses script direction, shape change, and sound-symbol trust early because those are the biggest early blockers for new learners.
- It uses diacritics as a scaffold before expecting confident unsupported reading. That matches the common pedagogical logic of helping learners decode accurately before removing support.
- It introduces visible usable language early. Strong beginner programs do not leave learners in abstract letter study for too long; they move into high-frequency words and fixed expressions quickly enough to create momentum.
- It delays broad grammar coverage and instead introduces small structural awareness at the point of need. That reflects good beginner design: notice a useful pattern, then use it, rather than front-loading grammar theory.
- It introduces short text handling only after the learner has some decoding ability, some known chunks, and some listening-recognition stability.
- It ends with transfer and self-learning habits. Authoritative beginner pedagogy aims not only at immediate memorization, but at giving learners a repeatable way to continue learning once teacher support becomes lighter.

The result is beginner pedagogy in the right order:

1. script and sound entry
2. supported decoding
3. first usable language
4. guided pattern awareness
5. very short text handling
6. independent beginner continuation

# 4. Free-vs-paid value logic

This plan is intentionally strong for free users, but it does not collapse paid value.

What free should clearly deliver:

- script fear reduction and first decoding confidence
- a small but meaningful starter set of words and expressions
- first listening trust
- first structural awareness
- first supported short-text win
- first self-learning transfer for a new word and a new phrase
- first understanding of review as part of retention

What remains intentionally reserved for paid expansion:

- much larger vocabulary fields and scene coverage
- broader conversation ability
- systematic grammar and morphology development
- wider reading range and longer texts
- deeper listening practice with more variation
- richer pronunciation coverage and contrast work
- longer-term review personalization and continuity

Commercially, the free track should create this exact learner reaction:

- "I really entered Arabic."
- "This app's method actually works."
- "I want the larger system, not just more of the same single lessons."

That is why the free track includes authentic wins, but keeps scale for paid:

- Free proves the learning loop.
- Paid scales the learning loop.

# 5. Risks and mitigations

- Risk: Stage A feels too abstract or too long.<br>Mitigation: every Stage A lesson still includes a visible Arabic success, not only explanations.
- Risk: Free users feel they learned methods but not "real Arabic."<br>Mitigation: Stage B creates early word and expression ownership, and every later study-skill lesson still teaches real Arabic content.
- Risk: Pattern awareness becomes grammar-heavy too early.<br>Mitigation: Stage C introduces only one manageable distinction at a time and keeps explanations light.
- Risk: The free track becomes so complete that paid value weakens.<br>Mitigation: the plan teaches entry, trust, and transfer, but not broad coverage, broad interaction, or systematic grammar depth.
- Risk: The final project feels like a test rather than empowerment.<br>Mitigation: Lesson 16 is framed as guided transfer using a tiny new task, not as a punitive summary exam.
- Risk: Learners still think review means failure.<br>Mitigation: Lesson 15 explicitly reframes review as memory stabilization and uses a visible "this got easier" experience.
- Risk: Later lesson generation drifts into overloaded lessons.<br>Mitigation: each lesson in this plan has exactly one core objective and a constrained target knowledge scope for downstream generation.

# 6. Suggested Codex execution order

Suggested implementation order for later lesson generation:

1. Generate Lessons 1-4 first as the common foundation layer.
   - These define beginner copy style, phonetics-support conventions, diacritics support, and V2 evidence expectations.
2. Generate Lessons 5-8 next.
   - These create the first visible product value and the first reusable review seeds for words, expressions, and audio recognition.
3. Generate Lessons 9-12 after that.
   - These require the earlier reading and recall loop to be stable before pattern awareness and short-text handling will feel natural.
4. Generate Lessons 13-16 last.
   - These depend on prior content being real enough for self-learning transfer and review logic to feel earned.
