# 首轮试听打分验收表 Batch 01

用途：

- 用于记录首轮试听 8 条样本的试听结果
- 先判断 slow 是否达标
- 样本达标后，再决定是否生成 normal 和是否扩展到 30 条

建议打分方式：

- 2 分：达标
- 1 分：基本可用，但需要微调
- 0 分：不达标，需要重生成

## 单项评分标准

| 项目 | 说明 |
| --- | --- |
| 发音正确 | 是否存在读错、漏读、长短音不清、基础音值错误 |
| 文本一致 | 实际播报是否与页面文本完全一致 |
| 节奏平缓 | 是否明显慢、稳，不急促 |
| 自然度 | 是否温和、清楚、自然，没有明显机器感 |
| 跟读友好 | 初学者是否容易听清并模仿 |
| 工程播放 | 放入工程后是否能正确命中并播放 |

## 通过建议

建议采用以下门槛：

- 单条总分至少 10 分，才算通过
- “发音正确” 不能低于 2 分
- “文本一致” 不能低于 2 分
- “工程播放” 不能低于 2 分
- 8 条里至少 7 条通过，才建议进入 normal 生成或批量扩展

## 复听建议

每条建议至少听 3 遍：

1. 第一遍只听发音对不对
2. 第二遍只听节奏和自然度
3. 第三遍放进工程里点播放，确认命中和实际跟读感受
